# Proposal: Update Theme and Add Content

## Why
User reported usability issues with the Brutalist theme:
- Buttons are missing text (contrast issue).
- Social icons are broken (visibility issue).
- Need a visual indicator for draft content.
- Missing core methodology documentation on the site.

## What Changes
1.  **CSS Enhancements**:
    - Fix button text contrast.
    - Fix social icon visibility.
    - Add "Draft" badge style.
2.  **Content Addition**:
    - Add OpenSpec methodology documents to the site.
    - Create a new "Methodology" sidebar section.

## Impact
- `src/styles/custom.css`: Style updates.
- `src/content/docs/methodology/`: New content.
- `astro.config.mjs`: Sidebar configuration.
