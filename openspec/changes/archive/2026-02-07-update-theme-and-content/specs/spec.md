# Specs

## UI Requirements

### Button Contrast
- **WHEN** a user views a button (primary action)
- **THEN** the text must be clearly visible against the background (high contrast)
- **AND** it should maintain the brutalist aesthetic (bold borders).

### Social Icons
- **WHEN** the header is viewed
- **THEN** social icons (GitHub etc.) must be visible (correct fill/stroke color).

### Draft Badge
- **WHEN** content is marked as draft
- **THEN** a "Draft" badge should be available (yellow/black warning style).

## Content Requirements

### Methodology Section
- **WHEN** the site is built
- **THEN** a new "Methodology" section should appear in the sidebar.
- **AND** it should contain:
    - `SDD_OpenSpec.md`
    - `SDD_SpecKit.md`
    - `SDD_Template.md`
