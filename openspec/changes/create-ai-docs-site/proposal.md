# Proposal: Create AI Docs Site

## 1. Обзор изменения (Change Overview)

### Зачем сейчас (Why Now)
*   Отсутствует единая база знаний по AI-проектам.
*   Информация разрознена, онбординг новых участников затруднен.
*   Нужен современный, быстрый ресурс для документации.

### Область и влияние (Scope & Impact)
*   **Новый компонент:** Сайт документации на базе **Astro Starlight**.
*   **Дизайн:** Тема в стиле **Neo-Brutalism** (Tailwind CSS).
*   **Контент:** Гайды, API References, Архитектура.
*   **Инфраструктура:** Статический сайт (SSG).

### Критерии успеха (Acceptance Criteria)
*   [ ] Инициализирован проект Astro Starlight.
*   [ ] Настроена тема (Brutalism: шрифты, цвета, границы).
*   [ ] Создана структура разделов (Sidebar).
*   [ ] Сайт собирается и работает локально.

### Не-цели (Non-Goals)
*   Бэкенд, базы данных, аутентификация (только статика).
*   Интеграция с LLM-чатом в первой версии.
