# Specs: AI Docs Site (Astro Starlight)

## 1. Пользовательские истории (User Stories)

### Читатель (Reader)
*   **Как** инженер, **я хочу** быстро находить нужную информацию через поиск или меню, **чтобы** не тратить время на навигацию.
*   **Как** новый сотрудник, **я хочу** видеть раздел "Getting Started", **чтобы** быстро погрузиться в контекст проектов.
*   **Как** пользователь с плохим зрением, **я хочу** иметь контрастный интерфейс (Brutalism style), **чтобы** текст был легко читаемым.

### Автор документации (Author)
*   **Как** технический писатель, **я хочу** писать документацию в Markdown, **чтобы** хранить её вместе с кодом в Git.
*   **Как** автор, **я хочу** использовать готовые компоненты (Note, Warning, CodeBlock), **чтобы** документация выглядела профессионально.

## 2. Сценарии (Scenarios)

### Scenario 1: Навигация по разделам
*   **Given** я нахожусь на главной странице сайта
*   **When** я кликаю на раздел "Guides" в боковом меню (Sidebar)
*   **Then** открывается список доступных гайдов
*   **And** URL меняется на `/guides`

### Scenario 2: Поиск информации
*   **Given** я нахожусь на любой странице
*   **When** я нажимаю `Ctrl+K` или кликаю на поле поиска
*   **Then** открывается модальное окно поиска (Pagefind)
*   **When** я ввожу "API"
*   **Then** я вижу список релевантных страниц с подсветкой найденного текста

### Scenario 3: Добавление новой статьи (Docs-as-Code)
*   **Given** я работаю в VS Code
*   **When** я создаю файл `src/content/docs/guides/new-guide.md`
*   **And** добавляю заголовок и контент
*   **Then** статья автоматически появляется в меню сайта (при локальной разработке)

## 3. Инварианты и ограничения (Invariants)

### Дизайн (Brutalism)
*   **Границы:** Все карточки и секции должны иметь жирные черные границы (`border-2` или `border-4` черного цвета).
*   **Тени:** Использовать жесткие тени без размытия (`box-shadow: 4px 4px 0px 0px #000`).
*   **Шрифты:** Использовать моноширинные шрифты для кода и "рубленые" (Sans-Serif) для заголовков.
*   **Цвета:** Высококонтрастная палитра (белый, черный, яркий акцентный цвет — например, неоновый зеленый или желтый).

### Технические ограничения
*   Сайт должен работать **без JavaScript** (основной контент должен быть доступен).
*   Отсутствие базы данных (всё хранится в `.md` файлах).
