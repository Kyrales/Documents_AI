# Welcome to AI Docs

This documentation site serves as the central knowledge base for our AI projects.

## Quick Start

To add a new page:
1. Create a markdown file in `src/content/docs/`.
2. Add frontmatter with `title` and `description`.
3. Commit and push.

## Tech Stack

- **Astro Starlight**: The documentation framework.
- **Tailwind CSS**: For styling.
- **Neo-Brutalism**: The design system.
